An E-Commerce Website with html , css and JavaScript

Features : 
- Home Page : load Data from  Api "https://afternoon-falls-30227.herokuapp.com/api/v1/products/"
- Contact us Page : to communicate with , and send data to server "https://afternoon-falls-30227.herokuapp.com/api/v1/contact_us"
- About Page : links to social media and some text
- Single Page : page for each product when you click on it from home page 
- cart page : contains all products added to cart
- add to cart : to add items in your shopping cart (you can add either from home page or single page product)
- remove product : remove item from cart page

icons used from Font-Awsome "https://stackpath.bootstrapcdn.com/font-awesome/4.7.0/css/font-awesome.min.css": 
- cart icon
- remove icon
